Python is a powerful programming language ideal for scripting and rapid application development. It is used in web development (like: Flask framework ), scientific and mathematical computing (like : NumPy , Tensorflow ) to desktop graphical user Interfaces (like : Pygame ).

Python has been the most loved programming language of 2019 , in fact most of the Students love it more than French .So lets see what makes Python so special!


It is Simple.
Python’s readability makes it a great first programming language — it allows you to think like a programmer and not waste time with confusing syntax. For instance, look at the above code to print “hello world” in Java and Python.


Runs Everywhere.

Python supports majority of the operating systems and can run on minimal hardware. Nowadays you will see Python is builtin in many of Linux distributions like Kali Linux , Raspbian , etc.




Huge Community
Python has a huge Community of developers helping each other out. So if you face any problem while doing a project, Just ask platforms like Stackoverflow !
